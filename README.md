# Post Management System (React CRUD App)

A **Post Management System** (Mini CRUD Blog Admin Panel) built with **React** and **Vite**, featuring full **Create, Read, Update, Delete (CRUD)** functionality. This project uses **localStorage** for data persistence and includes search, filter, and routing.

---

##  Tech Stack

- **React** (with Vite)
- **React Router DOM**
- **UUID** for unique post IDs
- **CSS** (can be swapped with Tailwind or Bootstrap)
- **localStorage** for storing posts
